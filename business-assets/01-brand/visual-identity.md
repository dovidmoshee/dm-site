# Calibrate Media Visual Identity

## Logo usage
- Primary: wordmark "Calibrate Media"
- Secondary: "CM" monogram
- Keep clearspace equal to the height of the C around the logo

## Color palette
- Ink: #0F172A
- Slate: #334155
- Mist: #E2E8F0
- White: #FFFFFF
- Signal Teal: #0D9488
- Soft Teal: #CCFBF1

## Typography
- Headline: Sora Semibold
- Body: Inter Regular
- Monospace: JetBrains Mono

## UI style rules
- Use rounded corners (16-24px)
- Use simple iconography with single-stroke lines
- Use whitespace generously
- Avoid dense text blocks

## Image direction
- Clean interfaces
- Real workflow screenshots
- Minimal visual clutter
- Team execution moments over stocky lifestyle shots
