# Cohevo Business Asset Kit

This folder contains the core assets to run Cohevo as a service business.

## How to use this kit

1. Replace placeholders in brackets (example: `[Client Name]`).
2. Start with brand and offer docs before publishing or selling.
3. Use sales and operations templates for every client.
4. Review legal docs with your attorney before use.
5. Track KPIs weekly using the finance templates.

## Folder map

- `01-brand/`: positioning, messaging, identity rules, and the Cohevo explainer
- `02-offers/`: offers, pricing, proposal template
- `03-sales/`: scripts, pipeline, outreach templates
- `04-operations/`: onboarding and delivery SOPs
- `05-marketing/`: website copy, social, case study templates
- `06-finance/`: invoice template, KPI dashboard, forecast
- `07-legal/`: legal checklist and draft templates
- `08-systems/`: stack blueprint and automation map template
- `notion/`: Notion workspace map and CSV imports
- `exports/docx/`: DOCX versions of all markdown assets
- `exports/pdf/`: PDF versions of all markdown assets

## Launch checklist

- Finalize offer and pricing
- Publish website copy
- Set up CRM pipeline
- Set up onboarding flow
- Set up invoicing and payment terms
- Finalize legal terms and privacy policy
- Start weekly KPI review cadence
