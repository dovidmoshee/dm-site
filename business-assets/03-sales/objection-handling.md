# Objection Handling - Calibrate Media

## "This is too expensive"
Response: Understood. We can reduce scope to the highest-impact workflow first and phase the rest.

## "We need to think about it"
Response: Makes sense. What specific concern should we resolve first: scope, timeline, or budget?

## "We can do this in-house"
Response: You probably can. The value here is speed, structure, and clean implementation without distracting your team.

## "Now is not the right time"
Response: Fair. What milestone or date would make this a priority so we can plan around it now?
