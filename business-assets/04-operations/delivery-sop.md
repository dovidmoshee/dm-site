# Service Delivery SOP - Calibrate Media

## Cadence
- Monday: Plan sprint and confirm priorities
- Tuesday-Thursday: Build and implement
- Friday: QA, documentation, client update

## Workflow
1. Capture requirement
2. Define outcome and acceptance criteria
3. Implement in staging or sandbox
4. QA against criteria
5. Document SOP and ownership
6. Roll out and monitor

## Quality checklist
- Owner assigned
- Trigger and output defined
- Error handling defined
- Documentation complete
- Client sign-off recorded
