# Client Onboarding Checklist - Calibrate Media

## Pre-kickoff
- Signed agreement received
- Initial payment received
- Internal project owner assigned
- Client workspace created
- Shared channel and docs folder created

## Access collection
- CRM access
- Project management tool access
- Analytics and reporting access
- Automation platform access
- Communication tools access

## Kickoff prep
- Confirm goals and success metrics
- Confirm in-scope workflows
- Confirm stakeholder list
- Confirm meeting cadence
- Confirm timeline and key milestones

## Week 1 outputs
- Current-state map
- Bottleneck list
- Priority workflow list
- Implementation sequence
