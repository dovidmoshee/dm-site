# Tech Stack Blueprint - Calibrate Media

## Core stack
- Website: Next.js
- CRM: HubSpot or Pipedrive
- Project management: ClickUp or Asana
- Knowledge base: Notion
- Automation: Zapier or Make
- Scheduling: Calendly
- Payments: Stripe
- Accounting: QuickBooks or Xero

## Minimum setup sequence
1. CRM pipeline with required fields
2. Lead form to CRM automation
3. Discovery and proposal templates
4. Contract + invoice workflow
5. Client onboarding workflow
6. Weekly reporting dashboard

## Naming conventions
- Workspaces: `Client - CompanyName`
- Automations: `[Process] - [Trigger] - [Action]`
- SOP docs: `SOP - [Process Name] - v1`

## Security baseline
- Use least privilege access
- Enable MFA on all tools
- Remove access within 24 hours of offboarding
